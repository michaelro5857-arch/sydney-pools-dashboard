# sydney-pools-dashboard
A real-time dashboard application for tracking Sydney pools data, built with Node.js, WebSockets, and Tailwind CSS.
# Live SDY - Real-Time Data Dashboard

A professional web application designed to fetch, process, and display real-time updates for Sydney pools data feed. 

##  Features
- **Real-Time Update:** Uses WebSockets for instant data broadcasting.
- **Responsive Design:** Fully optimized for mobile and desktop views using Tailwind CSS.
- **Data Caching:** Optimized backend fetching with Redis to reduce source server load.

##  Tech Stack
- **Frontend:** React.js, Tailwind CSS
- **Backend:** Node.js, Express
- **Database/Cache:** Redis
